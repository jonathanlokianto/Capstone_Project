# API Registrasi dan Login

API ini menyediakan endpoint untuk registrasi, login, dan pengelolaan pengguna menggunakan Node.js, Express, dan MySQL.

## Fitur

- Registrasi pengguna baru
- Login pengguna dengan token JWT
- Melihat daftar pengguna
- Menghapus pengguna berdasarkan ID

## Persyaratan

- Node.js
- MySQL
- npm (Node Package Manager)
